##into your philo folder:
  ```Bash
git clone git@github.com:AllanDantas21/philosophers_tester.git && cp philosophers_tester/test.sh . && rm -rf philosophers_tester
  ```

### Usage:
```Bash
./test.sh M -> to mandatory tests

./test.sh   -> to extra tests
  ```
